The TechCon Web Development Intensive is a 12-week program designed for beginners to become front-end developers. Through lectures, exercises, and projects, you'll learn HTML, CSS, JavaScript, web design principles, responsive design, and an introduction to front-end frameworks. Gain the skills and build a portfolio to launch your web development career. Experienced instructors, a supportive community, and project-based learning will equip you for success.
